# Лабораторная работа №1: Определение конфигурации ПК

## 📋 Описание
Целью данной работы является получение навыков определения аппаратной и программной конфигурации персонального компьютера с использованием специализированного программного обеспечения.

Тема: Определение конфигурации персонального компьютера перед проведением тестирования компьютерной системы.

## 🛠 Используемое оборудование и ПО
*   Персональный компьютер (ПК)
*   Программа для аудита системы: WinAudit (версия Freeware v3.2.1)

## 🚀 Порядок выполнения работы

1.  Запуск программы:
    *   Была запущена утилита winaudit.exe.
2.  Настройка параметров аудита:
    *   Через меню Вид -> Параметр были выбраны категории для сканирования:
        *   ✅ Обзор системы
        *   ✅ Операционная система
        *   ✅ Группы и пользователи
        *   ✅ Сеть TCP/IP
        *   ✅ Диспетчер устройств
        *   ✅ Видеоадаптер
        *   ✅ Версия BIOS
        *   ✅ Процессоры
        *   ✅ Память
        *   ✅ Жесткие диски
3.  Сканирование:
    *   Нажата кнопка "Применить" (или кнопка "АУДИТ"), после чего система была просканирована.
4.  Заполнение отчета:
    *   На основе полученного отчета WinAudit была заполнена таблица характеристик системы.
