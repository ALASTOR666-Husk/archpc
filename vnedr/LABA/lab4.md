# 🔧 Лабораторная работа №4  
## Диагностика работоспособности ПК с помощью PC Wizard

---

## 📋 Цель работы
Провести диагностику компьютера с помощью программы PC Wizard для получения подробной информации о конфигурации системы и тестирования производительности отдельных компонентов.

---

## 🎯 Задачи
- Получить обширную информацию обо всех установленных компонентах ПК
- Протестировать производительность процессора, памяти, видеокарты и дисков
- Выявить возможные дефекты оборудования
- Избежать потери данных благодаря своевременной диагностике

---

## 🛠 Используемое программное обеспечение

### PC Wizard 2014
Программа для комплексной диагностики компьютера и тестирования производительности.

Возможности программы:

📊 Информация о компонентах:
- Память (RAM)
- Материнская плата
- Устройства хранения данных
- Видеоподсистема
- Сетевые устройства
- Модемы, принтеры и др.

💻 Данные об ОС:
- Версия системы
- Установленные шрифты
- Библиотеки WinSock
- Активные процессы
- Модули и сервисы

⚡ Тестирование производительности:
- Процессор
- Жесткие диски
- CD/DVD-привода

Поддержка: 32- и 64-битные ОС

---

## 📝 Порядок выполнения работы

### 1. Подготовка
1. Запустить виртуальную машину Windows 7
2. Перенести файл pc-wizard_2014.2.14-setup.exe
   (программа не совместима с Windows 10)
3. Установить PC Wizard
4. Запустить программу
### 2. Запуск тестирования
- В левом меню выбрать опцию ТЕСТ
- Выбирать отдельные компоненты для тестирования

---
