# 🖥️ Лабораторная работа № 6
## Определение основных характеристик и тестирование видеосистемы ПК

---

### 📋 Описание
Данная лабораторная работа посвящена изучению современных видеокарт на графических процессорах **NVIDIA** и **AMD (ATI)**, а также технологиям их тестирования и мониторинга.

### 🎯 Цель работы
Изучить современные видеокарты, научиться снимать их характеристики и проводить стресс-тестирование видеосистемы персонального компьютера.

### ⏱️ Время выполнения
1 час

### 🛠️ Оборудование и Программное обеспечение
| Тип | Наименование |
| :--- | :--- |
| **Оборудование** | Учебный персональный компьютер |
| **ПО (ОС)** | Операционная система (Windows) |
| **Утилиты** | `GPU-Z` (информация), `FurMark` (стресс-тест), `FluidMark` (бенчмарк) |

---

### 📝 Ход работы

#### 1. Сбор информации о видеоадаптере (GPU-Z)
Необходимо скачать и установить утилиту **GPU-Z**. С помощью программы определить и зафиксировать следующие параметры:

- [ ] Имя графического адаптера (Intle(R) HD Graphics)
- [ ] Технологию выполнения (32 nm)
- [ ] Версию BIOS (2111 PC 14.34 02/23/2011 17:43:49)
- [ ] Поддерживаемый DirectX (10.1)
- [ ] Объем видеопамяти (N/A)
- [ ] Тип памяти (DDR3)
- [ ] Разрядность шины (128 Bit)
- [ ] Версию драйвера (9.17.10.4459 / Win 10 64)
- [ ] Частоту GPU (850 MHz)

#### 2. Стресс-тестирование (FurMark)
Запустить утилиту **FurMark** (версия v1.10.1 или выше) и провести тестирование в два этапа.

**Этап А: Тест с визуальными эффектами**
1. Настроить параметры тестирования согласно рекомендациям.
2. Запустить тест в режиме "бублик" (eye/donut mode).
3. Дождаться окончания тестирования (или 10 минут).
4. Записать результаты **FPS** (min:2; max:91; avg:11).
   > *Если тест не завершился за 10 минут, записать результат из левого верхнего угла и завершить тест клавишей `ESC`.*

**Этап Б: Тест без визуальных эффектов**
1. Запустить тест повторно.
2. Нажать `Пробел`, чтобы отключить отображение "бублика".
3. Пройти тестирование еще раз.
4. Записать результаты **FPS** (min:2; max:91; avg:19).

#### 3. Мониторинг производительности (FluidMark)
1. Установить и запустить программу **FluidMark**.
2. Установить необходимые параметры тестирования.
3. Запустить тест на **5 минут**.
4. Зафиксировать данные:
   - 📊 **FPS** (min=3; max=5; avg=3)
   - 🌡️ **Температура видеочипа**

#### 4. Анализ результатов
Сделать вывод о качестве установленного видеоадаптера на основе полученных данных (температура, стабильность FPS, характеристики).
