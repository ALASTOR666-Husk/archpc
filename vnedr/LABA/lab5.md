# 🖥️ Лабораторная работа №5  
## Тестирование качества монитора с помощью программных средств

---

## 📋 Цель работы
Провести диагностику монитора и выбрать оптимальные параметры цветового отображения с помощью специализированного программного обеспечения.

---

## 🎯 Задачи
- Изучить возможности программы MonitorTest
- Определить основные характеристики монитора
- Провести цветовое тестирование дисплея
- Проанализировать качество отображения различных цветов
- Выявить возможные дефекты монитора

---

## 🛠 Используемое программное обеспечение

### MonitorTest (PassMark Software)
Утилита для комплексного тестирования качества мониторов.

Возможности программы:

✅ Основные функции:
- Поддержка всех доступных разрешений и глубины цвета
- Тестирование контрастности
- Анализ цветопередачи
- Проверка отображения шрифтов
- Поддержка большинства типов мониторов

✅ Специальные тесты:
- Тест экранов Multitouch
- Проверка конвергенции
- Поддержка более 300 режимов работы видеокарт

---

## 📝 Порядок выполнения работы

### Часть 1: Сбор информации о мониторе

1. Установка программы:
  
   Запустить файл montest.exe
   
2. Определение характеристик монитора:
   
   Запишите следующие параметры:
   - 📺 Модель монитора
   - 📐 Разрешение (ширина × высота)
   - 📊 Соотношение сторон (4:3, 16:9, 16:10 и т.д.)
   - 🔌 Разъем подключения (VGA, DVI, HDMI, DisplayPort)
   - 📏 Размер экрана в дюймах
   - 🎨 Число отображаемых цветов
   - 📍 Плотность пикселей (PPI)

---


🔧 Основные характеристики

| Параметр | Значение |
|----------|----------|
| **Модель монитора** | VA2459-FHD |
| **Разрешение** | 1920×1080 |
| **Соотношение сторон** | 16:9 |
| **Разъем подключения** | VGA |
