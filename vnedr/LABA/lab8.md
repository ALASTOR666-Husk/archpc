# Лабараторная работа №8

На этой лабораторной работе мы занимались **тестированием персонального компьютера** с помощью утилиты **PassMark PerformanceTest**.

### Цель работы:
Провести финальное тестирование стабильности системы для последующей разработки технического задания по внедрению компьютерной системы.

### Что мы делали:

1. **Изучили возможности PerformanceTest** - программы для оценки производительности ПК, которая включает 27 стандартных тестов в 7 группах (тесты процессора, 2D/3D графики, дисков, памяти, CD/DVD-приводов)

2. **Провели 6 основных видов тестов:**
   - PassMark
   - CPUmark (включает 9 подтестов)
   - 2Dmark (7 подтестов)
   - 3Dmark
   - Memory Mark
   - Disk mark

3. **Сравнили результаты** с тестами из программы AIDA64 Extreme

4. **Контролировали температуру процессора** до и после теста CPUmark

5. **Дали субъективную оценку** каждому компоненту системы (от 1 до 10)

## Ответы на контрольные вопросы

### 1. Для каких целей может быть использована тестируемая компьютерная система?

Система (PassMark Rating: 505, 7% процентиль) подходит для:
- Офисных задач (документы, таблицы)
- Веб-серфинга и электронной почты
- Просмотра видео и прослушивания музыки
- Обучения и работы с простыми приложениями

**Не подходит** для: современных игр, видеомонтажа, 3D-моделирования.

---

### 2. Какие параметры компьютерной системы можно было бы улучшить?

**Приоритетные улучшения:**

1. **Видеоадаптер** (3D Graphics Mark: 64) — установить дискретную видеокарту (NVIDIA GTX 1650 или аналог)
2. **Процессор** (CPU Mark: 2269) — заменить на более производительный (Intel Core i5 / AMD Ryzen 5)
3. **ОЗУ** — увеличить до 16 ГБ
4. **Накопитель** — заменить HDD на SSD для ускорения системы
