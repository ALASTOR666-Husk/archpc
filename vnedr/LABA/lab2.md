---

## 📋 Цель работы
Оценить качество работы жесткого диска с помощью специальных программ для диагностики и тестирования.

---

## 🛠 Используемое программное обеспечение

### 1. CrystalDiskMark
Небольшое приложение для измерения скорости записи и чтения устройства.

Возможности:
- Измерение скорости чтения/записи
- Тестирование с разными размерами блоков
- Настраиваемое количество тестов
- Выбор объема данных для теста

### 2. CrystalDiskInfo
Программа для считывания SMART-данных жесткого диска.

Возможности:
- Мониторинг температуры устройства
- Определение количества проблемных секторов
- Анализ ошибок позиционирования считывающей головки
- Время раскрутки диска
- Версия прошивки
- Статистика использования диска

---

## 📝 Порядок выполнения работы

### Этап 1: Тестирование скорости (CrystalDiskMark)

1. Установить программу CrystalDiskMark
2. Запустить программу
3. Настроить параметры тестирования:
   - Количество запусков тестов: 6
   - Объем тестируемых данных: 1000 MB
   - Выбрать раздел жесткого диска
4. Нажать кнопку All для запуска всех тестов
5. Оформить результаты в виде таблицы:

| Вид теста | Чтение (MB/s) | Запись(MB/S) |
| :--- | :--- | :--- |
| SEQ1M Q8T1 | 441.46 | 374.11 |
| SEQ1M Q1T1 | 401.00 | 354.43 |
| RND4K Q32T1 | 30.09 | 97.60 |
| RND4K Q1T1 | 23.32 | 60.14 |

### Этап 2: Диагностика состояния (CrystalDiskInfo)

Записать следующие параметры:
- 🌡️ Температура диска
- ✅ Техническое состояние
- 🔄 Число включений
- ⏱️ Общее время работы
- 💫 Скорость вращения (если имеется)
